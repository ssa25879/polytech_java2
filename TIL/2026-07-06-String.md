# 2026-07-06 문자열

## 문자열 처리

### 결합

- `+` 기호를 사용하여 문자열을 결합할 수 있음.
    - stack 단에 있는 메모리 주소를 계속하여 갱신하고, 기존에 Heap 단에 있는 메모리를 가비지컬렉터가 처리함을 반복함
    - `System.out.println("Hello" + " World!")` 출력시 "Hello World!" 출력됨
    - `+` 를 통해 결합 할 때 마다 새로운 String을 생성하여 붙이기에 시간이 오래걸림
        - String 인스턴스는 불변 객체이기 때문
- StringBuilder 방식
    - `append((문자열))` 메서드를 통해 문자열을 결합한 결과를 버퍼에 담아두고 `toString()` 을 통해 결과를 얻음

### 일부만 떼어내 출력

- `substring((잘라내기 시작 부분),(잘라내는 인덱스 번호 - 포함하지 않음))` 사용하여 떼어내어 출력 할 수 있음
- `String s1 = "Hello"` 일 때 `System.out.println(s1.substring(0, 2));`를 실행하여 "He"만 출력할 수 있음

### 치환

- `replace((치환당할 문자열), (치환되는 문자열))` 을 통해 문자열 내의 문자나 문자열을 치환할 수 있음
- `String s1 = "Hello"` 일 때 `System.out.println(s1.replace("ll", "xx"));`를 통해 출력값을 "Hexxo"로 변환 가능

### 분리

- 문자열 내의 기호, 문자 등을 기준으로 문자열을 분리할 수 있음
- `String s1 = "1,2,3"` 일 때 `String split[] = s1.split(",")` 를 통해 `,` 를 기준삼아 나눌 수 있음
    - 출력 시 split[0] = "1" , split[1] = "2", split[2] = "3" 이 출력됨

### 변환

- 대문자는 `toUpperCase()` 소문자는 `toLowerCase()` 를 사용하여 변환 가능
- `trim()` 을 통해 문자열 앞/뒤에 있는 공백을 지울 수 있음

### 검색

- 문자열에 문자나 문자열이 몇번째에 위치하여 있는지 검색할 수 있음
    - `indexOf((검색할 문자))` 를 사용해 검색
    - `String s1 = "Hello"` 일 때 `System.out.println(s1.indexOf('e'));` 실행 시 1이 출력 됨
    - `String s1 = "Hello Java"` 일 때 `System.out.println(s1.indexOf("Java"));` 실행 시 "Java" 문자열의 위치인 6이 출력 됨
- 문자열에 다른 문자열이 포함되어 있는지 검색할 수 있음
    - `contains((검색할 문자열))` 을 사용해 검색 출력은 참/거짓
    - `String s1 = "Hello Java"` 일 때 `System.out.println(s1.contains("Java"));` 실행시 true 출력함
- 문자열이 끝나는 단어가 맞는지 확인할 수 있음
    - `endsWith((문자열))` 을 사용해 해당 문자열로 끝나는지 확인 할 수 있음
    - `String s1 = "Hello Java"` 일 때 `System.out.println(s1.endsWith("Java"));` 실행시 true 출력함
- 문자나 문자열을 뒤에서 부터 검색하여 앞에서 몇번째 인덱스 번호에 위치하였는지 확인할 수 있음
    - 뒤에서 부터 검색하므로 "Java Hello Java" 문자열에서 "Java"를 검색하면 뒤에 있는 Java의 시작위치인 11을 출력함
    - `lastIndexOf((문자열))` 을 사용해 뒤에서 몇 번째에 위치하였는지 알 수 있음
    - `String s1 = "Hello Java"` 일 때 `System.out.println(s1.lastIndexOf("Java"));` 실행시 "Java" 문자열의 시작위치를 출력함

### 내용 비교

- `String s1 = "Java"` `String s2 = "java"` 둘이 있을 때를 예시로 사용
    - `s1.equals(s2)` 를 통해 내용을 비교할 수 있음
    - `s1.equalsIgnoreCase(s2)` 를 통해 대소문자를 무시하고 내용을 비교할 수 있음

### 문자열 길이

- `length()` 를 통해 문자열의 길이를 출력 할 수 있음
- `isEmpty()` 를 통해 길이가 0인지 아닌지 참/거짓 여부를 알 수 있음

### 과제 중 알게된 점

- 일부만 떼어내어 출력하기 위해 `substring()` 을 사용중 IDE에서 `charAt()` 사용을 추천받아 사용함
    - `substring()`의 경우 여러 문자를 떼어낼 때 주로 사용
        - 잘라낸 문자열로 새로운 `String` 객체를 만들어서 반환
    - `charAt()`의 경우 하나의 문자만을 떼어낼 때 사용
        - `charAt()` 으로 찾아낸 문자는 String 객체를 만들어서 반환하는 것이 아님
        - 값은 `char` 로 반환
